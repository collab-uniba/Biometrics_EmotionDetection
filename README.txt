In order to use this script:
- RSTUDIO REQUIRED
- Copy the folder "emotion_detection" in C:/
- Each directory has an install.R which you need to run in order to have all library required
- Each directory has its own main (main.r for feature_extraction, main_tuning.R for tuning, evaluation.R for mcnemar)
- Each directory has an appropriate folder in which insert input (Input for feature_extraction must to be insert in directory "data_subjects")

- Each input file need to be declared in the input vector of its appropriate main
